Robotics Programs

The program will take place every Saturday however when school closes the program will run from Monday- Saturday until the reopening of schools. The program will run for 3 hours a day session which will be from 9am to 12pm.

Purpose: You will assemble and program the robot to detect and avoid obstacles around the area

  • Electronic components are the parts used in robot to carry information and help in the function of the robot. In this subject, you will learn the following components:
    • Arduino
    • Ultrasonic sensor
    • Resistor
    • Motors
    • Motor shield
    • LEDs
  • Basic programming is the process of writing instructions that direct the computer to perform a set of tasks. The programme will be using C++ to code the arduino. The fundamental programming concepts to learn are:
    • Variable Declaration
    • Control Structure
    • Data Structure
    • Object-oriented programming
    • Debugging
    • programming tools
  • Block coding turns programming into a drag-and-drop process by converting text based code into visual blocks. Each block contains real code and when they’re combined together, they create curtain function. The types of blocks can include:
    • Motion Blocks
    • Loop Blocks
    • Event Blocks
    • Sound Blocks
    • Visual Effect Blocks
